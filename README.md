# al-many-class
**Requirements**
* Python 3.7
* fastText
* Pytorch >= 1.3.0
* Pytorch_transformers (also known as transformers) 2.3.0

**How to run**
```
python run_ft.py --dataset tnews_tokenized --number -1 --mode 1 --factor_mode 4
python run_bert.py --dataset tnews_tokenized --number -1 --mode 1 --factor_mode 4
```
